# Data Engineer

## Role Overview
The Data Engineer is responsible for designing, building, and maintaining the
infrastructure and pipelines that collect, store, and process data. They ensure
that clean, reliable data is available for analytics, reporting, and machine
learning workloads.

## Responsibilities
- Design and implement data pipelines (ETL/ELT)
- Build and maintain data warehouses and data lakes
- Ensure data quality, consistency, and availability
- Optimize data storage and processing performance
- Collaborate with Data Scientists to prepare data for modeling
- Monitor and maintain pipeline reliability

## Interacts With
- Backend Analyst (data sourcing and storage)
- Data Scientist (data preparation and feature engineering)
- Infrastructure Analyst (cloud and compute resources)
- Tech Lead (architecture alignment)

## Key Skills
- Proficiency in Python, SQL, and data processing frameworks (Spark, dbt, etc.)
- Experience with cloud data platforms (BigQuery, Redshift, Snowflake)
- Pipeline orchestration tools (Airflow, Prefect)
- Data modeling and warehouse design
