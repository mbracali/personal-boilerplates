# Data Scientist

## Role Overview
The Data Scientist is responsible for extracting insights from data and building
predictive or generative models that drive business decisions and power
intelligent features in the product.

## Responsibilities
- Explore and analyze datasets to identify patterns and insights
- Build, train, and evaluate machine learning models
- Collaborate with Data Engineers on feature engineering and data preparation
- Translate business problems into analytical and modeling approaches
- Communicate findings clearly to non-technical stakeholders
- Monitor model performance in production and retrain as needed

## Interacts With
- Data Engineer (data availability and quality)
- Backend Analyst (model serving and integration)
- Product Owner (business problem definition)
- Tech Lead (technical feasibility and deployment strategy)

## Key Skills
- Proficiency in Python and ML frameworks (scikit-learn, PyTorch, TensorFlow)
- Statistical analysis and mathematical modeling
- Data visualization and storytelling
- Familiarity with MLOps practices
