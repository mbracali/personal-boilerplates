# Tech Lead

## Role Overview
The Tech Lead is the most senior technical figure on the team. They are
responsible for the overall technical direction of the project, ensuring
code quality, architectural decisions, and alignment between all technical
disciplines.

## Responsibilities
- Define and enforce the technical architecture
- Make critical technology stack decisions
- Review code and ensure quality standards
- Unblock developers facing complex technical challenges
- Bridge communication between technical team and Product Owner/PM
- Mentor junior and mid-level developers

## Interacts With
- Frontend and Backend Analysts (technical guidance)
- Data Eng and Data Scientist (architecture alignment)
- Infrastructure Analyst (deployment and scalability decisions)
- Project Manager (effort estimation and technical risk assessment)

## Key Skills
- Deep expertise in software architecture
- Broad knowledge across frontend, backend, and infrastructure
- Leadership and mentoring
- Strong communication skills
