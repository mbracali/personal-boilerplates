# Frontend Analyst

## Role Overview
The Frontend Analyst is responsible for building and maintaining the user-facing
layer of the application. They translate UI/UX designs into functional,
accessible, and performant interfaces.

## Responsibilities
- Implement user interfaces from design specifications
- Ensure responsiveness and cross-browser compatibility
- Integrate with backend APIs and services
- Optimize frontend performance and accessibility
- Write unit and integration tests for UI components
- Collaborate with UX/UI designers to refine user experience

## Interacts With
- Backend Analyst (API contracts and integration)
- Tech Lead (architecture and code review)
- Product Owner (feature clarification)

## Key Skills
- Proficiency in HTML, CSS, JavaScript and modern frameworks (React, Vue, etc.)
- Understanding of REST and GraphQL APIs
- Performance optimization techniques
- Accessibility standards (WCAG)
