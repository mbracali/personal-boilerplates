# Backend Analyst

## Role Overview
The Backend Analyst is responsible for building and maintaining the server-side
logic, APIs, databases, and integrations that power the application. They ensure
the system is reliable, secure, and scalable.

## Responsibilities
- Design and implement REST or GraphQL APIs
- Build and maintain business logic and data models
- Integrate with third-party services and external APIs
- Ensure data security, validation, and error handling
- Write unit and integration tests
- Optimize query performance and application throughput

## Interacts With
- Frontend Analyst (API design and integration)
- Data Engineer (data pipeline and storage design)
- Infrastructure Analyst (deployment and scaling)
- Tech Lead (architecture decisions)

## Key Skills
- Proficiency in backend languages and frameworks (Python, Node.js, etc.)
- Database design (SQL and NoSQL)
- API design best practices
- Security fundamentals (authentication, authorization)
