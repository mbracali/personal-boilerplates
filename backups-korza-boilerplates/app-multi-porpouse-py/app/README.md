# Application Code

All application functions and core logic are located in this folder. Everything outside this directory is related to environment mechanisms, settings, or tests.

## 📂 Folder Structure

### `/core`
Core functions of the application. This is where the main business logic and essential functionality should be implemented.

### `/libs`
Full libraries that can be used to help and extend the application's functionality. These are reusable components and modules that provide additional capabilities to your app.

### `/pages`
Created primarily for Streamlit packages, but can be customized as needed. This folder should store all visual components and user interface elements.

### `/utils`
Utility modules and functions for the application. Helper functions, common utilities, and shared code that supports the core functionality.

---

*Keep your application code organized and modular by following this structure.*
